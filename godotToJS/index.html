Godot에서 JavaScript로
구현하기 앞서, 간단하게 메인씬을 만들겠습니다.

 이 문서는 Chromium기반 웹 브라우저를 기준으로 합니다.

 들어가기에 앞서 간단하게 사용법을 익히기 위해, 엔진을 시작하면 처음 배우는 캐릭터 움직임을 활용할 예정입니다.
 먼저 노드 구조를 다음과 같이 배치해주세요.

Node2D(MainScene)
ㄴ CharacterBody2D
 ㄴCollisionShape2D
 ㄴSprite2D
 ㄴVisibleOnScreenNotifier2D

 위의 캡쳐와 같이 노드를 배치하셨다면 CharacterBody2D에 스크립트를 추가해주세요.

 캐릭터바디에 사용자의 방향키 입력에 따른 이동기능을 추가함과 동시에 웹에 해당 캐릭터의 위치와 플레이어의 입력을 출력하는 기능을 만들 것입니다. 아래와 같이 스크립트를 추가해주세요.

var current_velo:Vector2 = Vector2.ZERO
var spd:float = 500.
 
var input:Vector2
 
func _physics_process(delta: float) -> void:
    get_input()
 
    if input != Vector2.ZERO:
        current_velo = lerp(current_velo, input * spd, .155)
    else:
        current_velo = lerp(current_velo, Vector2.ZERO, .155)
 
    velocity = current_velo

 방향키를 눌렀을 때, 캐릭터가 이동하는지 확인하세요. (F5 또는 F6을 눌러 실행시킬 수 있습니다.)

 완료되었다면 다음으로 JavaScriptBridge 클래스를 이용하여 자바스크립트의 변수나 메서드들을 GDScript로 불러와보겠습니다.

 웹으로 내보내진 프로그램은 자바스크립트가 콘솔로 메세지를 내보내는 것과 같은 경로로 메세지를 내보냅니다. 개발자도구를 열어보는 것으로 엔진과 자바스크립트가 발생시키는 오류와 경고를 모두 볼 수 있습니다.
 또는 사용자가 고도엔진에서 window객체의 alert(msg), console의 log(msg)를 쓸 수 있습니다.
 window와 console객체를 GDScript에서 JavaScriptObject로 정의하면 해당 객체를 엔진 내에서 참조할 수 있습니다.
