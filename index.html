JavaScriptBridge

이 문단은 공식문서에 서술된 JavaScriptBridge 클래스 설명의 보완입니다.

Godot Engine 공식 문서의 Export for the Web 한 번 읽어주시길 바랍니다.

자바스크립트와 고도는 모두 동적타입 언어이지만 자바스크립트와 달리 GDScript는 정적타이핑 작성을 적극적으로 권장하며 성능에 영향을 미칩니다.

아래는 자바오브젝트에서 value을 가져왔을 시 GDScript에서 대응되는 타입입니다.

  • Boolean > bool
  • Number > int / float
  • String > String
  • ArrayBuffer, TypedArray > PackedByteArray

그 밖의 함수는 Callable로 대응됩니다.
