GDScript와 JavaScript 테스트 환경 구축

@GDScript 2.0와 JavaScript

 이 문서는 공식문서에 서술된 JavaScriptBridge 클래스 문서를 보충설명 및 응용하는 방법을 알리기 위해 만들었습니다.

 Godot Engine을 웹으로 내보내는 방법은 공식 문서의 Export for the Web 한 번 읽어주시길 바랍니다.

 자바스크립트와 고도는 모두 동적 언어이지만 자바스크립트와 달리 GDScript는 정적타이핑 작성을 적극적으로 권장하며 성능에 영향을 미칩니다.
 이에 비해, 자바스크립트는 사용자가 타입을 지정하는 방법이 마땅히 없으며, 필요하지도 않습니다.

아래는 자바오브젝트에서 값(value)을 가져왔을 시 GDScript에서 대응되는 타입입니다.

  • Boolean > bool
  • Number > int / float
  • String > String
  • ArrayBuffer, TypedArray > PackedByteArray

 함수는 Callable로 대응됩니다. 만약 GDScript가 JS에서 해당 타입의 오브젝트들을 불러오게 된다면 대응되는 타입을 반환하게 됩니다.

from http import server
 
class HTTPServer(server.SimpleHTTPRequestHandler):
    def end_headers(self):
        self.send_my_headers()
        server.SimpleHTTPRequestHandler.end_headers(self)
 
    def send_my_headers(self):
        self.send_header("Access-Control-Allow-Origin", "*")
        self.send_header("Cross-Origin-Embedder-Policy", "require-corp")
