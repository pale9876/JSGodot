HTML5 템플릿 생성

 프로젝트 창, 상단 MenuBar에서 프로젝트 > 내보내기를 클릭하시면 다음과 같은 창이 뜨게 됩니다.

 여기서 저희가 핵심적으로 알아야 할 기능들은 다음과 같습니다.

Export Path: 프로젝트를 내보낼 경로를 설정합니다.

Custom HTML Shell: 사용자가 만든 또는 지정한 HTML 문서 양식으로 내보냅니다.

Head Include: 내보낼 때에 head태그에 적용합니다. 주로 스크립트와 css경로를 삽입할 수 있습니다.

  간단한 기능을 삽입할 때에는 Head Include를 이용해도 좋지만, 같은 프로젝트에서 서로 다른 프로그램을 만들어 내보낼 때에는 'Custom HTML Shell'은 유용합니다.
