How to Install Godot

GodotEngine 초심자를 위한 설치가이드 문서입니다.

Installation

 먼저 GodotEngine을 설치하고 Project까지 간단하게 알아보겠습니다. GodotEngine 홈페이지에서 고도엔진을 다운로드 받으세요.

 이곳에서 다운로드를 완료하셨다면, 압축을 풀고 exe파일을 실행시키면 됩니다.
 Godot Engine - .NET 버전은 웹으로 내보내기를 지원하지 않지만, 이후에 업데이트되는 버전에서 지원이 약속되어 있습니다.

 4.x .NET 버전은 웹 내보내기가 불가능합니다

Project Settings

 실행까지 완료하셨다면 `새로 만들기`를 클릭하신 후, 프로젝트를 만들어주세요. 초기 프로젝트 설정을 `호환성`으로 맞춰주세요. 호환성으로 설정하지 않으면 구동되지 않습니다.

프로젝트명과 경로는 한글이 포함되어도 무관합니다.
